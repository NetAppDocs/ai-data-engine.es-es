= Inicio rápido para AI Data Engine
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Para poner en marcha tu sistema AI Data Engine, necesitas instalar tus componentes de hardware, configurar tu clúster, configurar el acceso a los datos desde tus hosts al sistema de almacenamiento y aprovisionar tu almacenamiento.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["1"] ¿Se instalará AIDE con un nuevo o existente cluster AFX?
[role="quick-margin-para"]
Tienes que decidir si AIDE y AFX se instalarán juntos al mismo tiempo o si AIDE se integrará con un clúster AFX existente.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Dos"] Instala y configura tu hardware
[role="quick-margin-para"]
link:../install-setup/aide-hardware-requirements.html["Instala y configura"] tus nodos de computación del cluster AIDE. Dependiendo del entorno de instalación, también asegúrate de https://docs.netapp.com/us-en/ontap-afx/install-setup/install-setup-workflow.html["instalar"^] el hardware AFX.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Tres"] Configura tu clúster
[role="quick-margin-para"]
Usa ONTAP System Manager para guiarte en un proceso rápido y sencillo para link:../install-setup/cluster-setup-afx.html["Configura AIDE con un clúster AFX"].

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Cuatro"] Configura los espacios de trabajo y el acceso a los datos
[role="quick-margin-para"]
link:../workspaces/create-workspace.html["Configura un espacio de trabajo y usuarios para ese espacio de trabajo que puedan acceder a los datos de AI Data Engine"].

.¿Qué sigue?
Ahora puedes usar ONTAP System Manager para gestionar tu AI Data Engine y ayudar a que tus ingenieros de datos y científicos de datos comiencen con sus espacios de trabajo y configuraciones.
