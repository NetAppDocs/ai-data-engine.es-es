= Instala el sistema de almacenamiento AFX para AI Data Engine
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Instala el sistema de almacenamiento AFX como primer paso en el despliegue de AI Data Engine (AIDE). El sistema de almacenamiento AFX proporciona la infraestructura de almacenamiento y es necesario antes de instalar los nodos de cómputo de datos.

Sigue el link:https://docs.netapp.com/us-en/ontap-afx/install-setup/install-setup-workflow.html["Documentación de instalación de AFX 1K"^] para instalar el sistema de almacenamiento AFX.

.Qué sigue
Después de completar la instalación del sistema de almacenamiento AFX, link:install-workflow.html["instala los Data Compute Node"].
