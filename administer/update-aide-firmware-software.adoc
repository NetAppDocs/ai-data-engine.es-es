= Actualizar AI Data Engine software
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Como administrador de almacenamiento, puedes actualizar el software AI Data Engine (AIDE), el firmware NetApp DCN y otros archivos de sistema en tu sistema AIDE usando ONTAP System Manager.

AIDE no soporta la funcionalidad de actualizaciones automáticas de software de ONTAP. Puedes registrarte para recibir notificaciones de link:https://mysupport.netapp.com/site/downloads["NetApp Support Site descargas"^], pero todas las actualizaciones del software de AIDE se hacen manualmente por el admin.

.Acerca de esta tarea
El paquete de software AIDE combinado es significativamente más grande que un paquete de actualización típico de ONTAP (el paquete AIDE es de aproximadamente 40GB). Planifica tiempos de carga e instalación más largos cuando actualices el software AIDE.

.Antes de empezar
* Necesitas privilegios de _administrador de almacenamiento_ para actualizar el firmware de NetApp DCN y el software AIDE.
* Necesitas tus credenciales de NetApp Support Site para tu cuenta activa.
* link:upgrades-aide-summary.html#compatibility-matrix["Garantiza la compatibilidad entre ONTAP, el firmware NetApp DCN y el software AIDE al planificar las actualizaciones"].



IMPORTANT: La funcionalidad de revertir no es compatible con el firmware DCN o las actualizaciones de software AIDE de NetApp. Después de instalar una actualización, no puedes revertir a una versión anterior.

.Pasos
. https://mysupport.netapp.com/site/products/all/details/aide/downloads-tab["Descarga el archivo combinado de actualización de firmware de NetApp DCN y software AIDE a un cliente local"^].
. En System Manager, selecciona *Cluster > Configuración > Actualizaciones de software*.
. Junto a *Actualizaciones de software*, selecciona image:icon_arrow.gif["flecha azul apuntando a la derecha"].
. En actualizaciones de AI Data Engine, selecciona *Añadir archivos de software de AI Data Engine* y selecciona el paquete de actualización.
. Una vez finalizada la carga de paquetes, selecciona *Update* para comenzar a instalar las actualizaciones en las DCN de NetApp.
+

NOTE: La Consola AIDE no está disponible o accesible cuando la actualización de NetApp DCN está en curso.



.Resultado
NetApp los DCN se actualizan con el software AIDE y se muestra la versión actualizada de cada nodo.
